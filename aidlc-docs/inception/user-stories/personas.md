# 테이블오더 서비스 - 페르소나

## Persona 1: 고객 (Customer)

| 항목 | 내용 |
|------|------|
| 이름 | 김민수 (대표 고객) |
| 역할 | 매장 방문 고객 |
| 연령대 | 20~50대 |
| 기술 수준 | 스마트폰/태블릿 기본 사용 가능 |

**목표:**
- 대기 없이 빠르게 주문하고 싶다
- 메뉴를 편하게 탐색하고 선택하고 싶다
- 주문 상태를 확인하고 싶다

**동기:**
- 직원 호출 없이 자유롭게 주문
- 주문 실수를 줄이고 싶음 (장바구니에서 확인 후 주문)

**불편사항:**
- 복잡한 로그인 절차
- 느린 화면 반응
- 주문 후 상태를 알 수 없는 상황

---

## Persona 2: 매장 관리자 (Admin)

| 항목 | 내용 |
|------|------|
| 이름 | 박서연 (대표 관리자) |
| 역할 | 매장 운영자 / 점주 |
| 연령대 | 30~60대 |
| 기술 수준 | 기본 PC/태블릿 사용 가능 |

**목표:**
- 들어오는 주문을 실시간으로 파악하고 싶다
- 테이블별 주문 현황을 한눈에 보고 싶다
- 매장 운영을 효율적으로 관리하고 싶다

**동기:**
- 인건비 절감 (주문 접수 인력 감소)
- 주문 누락 방지
- 테이블 회전율 향상

**불편사항:**
- 주문이 밀릴 때 놓치는 경우
- 테이블별 정산이 복잡한 상황
- 이전 고객 주문과 현재 고객 주문이 섞이는 문제
